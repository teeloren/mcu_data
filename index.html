Marvel Cinematic Universe Box Office Dataset

Utilized python libraries' pandas and matplotlibto make calculations and visualizations from Marvel movies box office data.

Project Description

The dataset represent box office data for marvel movies like the release date, tomato meter, audience score, movie duration, producution budget, and domestic and worldwide box office ticket sales.

  • Dataset consisted of 27 rows and 10 columns.
  • Dataset was checked for null and duplicate values. None found.
  • 3 columns types were changed from type object to type float (production_budget, opening_weekend, domestic_box_office).
